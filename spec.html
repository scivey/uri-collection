PredRouter

  • has #handle and #use methods
  • accepts and uses a predicate-handler pair
  • chooses the route based on predicate result, not on sequence alone
  • chooses the first true-returning route when later route predicates would return true for the pipeline element(s)
  • allows for a flexible number of pipeline elements (handles different routing function arities)
