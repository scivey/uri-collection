should

  • should return true for URI instances.
  • should not return true for strings.
  • should not return true for other constructed objects.

map

  • should return a new URICollection when a map operation returns URIjs instances.
  • should return a vanilla array when a map operation returns results other than URIjs instances.
  • should not mutate URIjs instances in the original collection
  • should produce the same result as a series of individual URIjs instance manipulations.

toArray

  • should return a vanilla javascript array
  • should return *clones* of its internal URIjs instances

clone

  • should return a new URICollection instance
  • should create a copy with the same number of elements
  • should create a copy with individual URIjs elements cloned themselves
  • should create a copy with DEEP-equal elements

each

  • should return a new URI collection, even without an iterator return value
  • should not mutate URIjs instances in the original collection
  • should produce the same result as a series of individual URIjs instance manipulations.

pluck

  • should extract the given property from each URI instance.
  • should return an array with length equal to that of the collection.

invoke

  • should return the same number of results as there are elements in the original collection.
  • should return a vanilla array when passed a method name but no additional parameters.
  • should return a new URICollection when passed a method name and additional parameters.
  • when called with only a method name, should return the same array of results returned from manual iteration + no-arg invocation over a plain array of URIjs instances.
  • when called with a method and additional parameters, should return a collection of URIjs instances deep-equal to the manual-iteration equivalent.

some

  • should return true if any collection elements pass the truth test
  • should return false if zero collection elements pass the truth test

every

  • should return true if all collection elements pass the truth test
  • should return false if any collection elements fail the truth test

contains

  • when passed a string, returns true if the string is equal to the `.toString()` result of any element.
  • when passed a string, returns false if the string is not equal to the `.toString()` result of any element.
  • when passed a URLjs instance, returns true if the `.toString()` of that instance equals the `.toString()` of any element.
  • when passed a URLjs instance, returns false if the `.toString()` of that instance does not equal the `.toString()` of any element.
  • when passed a RegExp instance, returns true if calling that RegExp's `#test()` with any element's `.toString()` value returns true.
  • when passed a RegExp instance, returns false if calling that RegExp's `#test()` with every element's `.toString()` value returns false.

reduce

  • should not wrap its output in a URICollection.
  • should return the same result as iteration over a plain array of the same URIs.
  • should return correct results.

reduceRight

  • should not wrap its output in a URICollection.
  • should return the same result as iteration over a plain array of the same URIs.
  • should return correct results.
  • should return different results than `reduce` when operation is non-associative

find

  • should return a URIjs instance.
  • should return the first matching instance
  • should return null if no match is found

findWhere

  • should return a URIjs instance.
  • should convert a key-val query into a function invoking the correct instance method
  • should return the first matching instance
  • should return null if no match is found

where

  • should return a URICollection instance.
  • should convert a key-val query into a function invoking the correct instance method
  • should return _every_ matching URIjs instance.
  • should return _only_ matching URIjs instances.
  • should return an empty URICollection instance if no matches are found.

sample

  • should return a URIjs instance when called with no parameters or with n == 1.
  • should return a URICollection instance when called with n > 1.
  • should return a URICollection with size == n when n > 1.

filter

  • should return a URICollection instance.
  • should return all matches from the collection
  • should return the same result (count + deep-equal) as manual iteration over a plain array of the same URIs

reject

  • should return a URICollection instance.
  • should return all matches from the collection
  • should return the same result (count + deep-equal) as iteration over a plain array of the same URIs

at

  • should return a URIjs instance
  • should return a URIjs instance with a `.toString()` value equal to the string value at index `i` of the collection's `.toArray()` output.
  • should return null if there is no URL instance at that index.
  • should return the (N - 1)th from the last element if a negative index is passed.

first

  • if N==null or N==1, should return a URIjs instance
  • if N>1, should return a URICollection instance.
  • if N>1, should return a collection with N elements.
  • should return the first N elements

last

  • if N==null or N==1, should return a URIjs instance
  • if N>1, should return a URICollection instance.
  • if N>1, should return a collection with N elements.
  • should return the last N elements

rest

  • should return a URICollection instance for any N, including null
  • if n is null, should return a number of elements equal to `size() - 1`
  • if n is NOT null, should return a number of elements equal to `size() - n`
  • should return the elements from index N to the end.

initial

  • should return a URICollection instance for any N, including null.
  • if n is null, should return a number of elements equal to `size() - 1`
  • if n is NOT null, should return a number of elements equal to `size() - n`
  • should return the elements from index 0 to (size - (n+1)).

countBy

  • should accept a property string and use the matching property as the counting criteria
  • should accept a function and use it to create the counting criteria
  • should return a total count equal to the number of instances in the collection

groupBy

  • should accept a property string and use the matching property as the grouping criteria
  • should accept a function and use it to create the grouping criteria
  • should have the same number of keys as the number of unique results for the grouping function over a plain array of the collection's elements.
  • should have the same total number of URIs as the original collection

sortBy

  • should accept a property string and use the matching property as the sorting criteria
  • should accept a function and use it to create the sorting criteria
  • should return an instance of URICollection
  • should return a collection with the same number of elements as the original.

shuffle

  • should return an instance of URICollection
  • should return a collection with the same number of elements as the original
  • should return a collection with elements in a different order
