add(linkList)

Add URIs to the collection.

Parameters
  • linkList: String, Object, Array

    A single link, multiple links, or an array of links. A given link can be an existing URIjs instance or a string, which will be converted into a URIjs instance.

Returns
  • URICollection

    A reference to the same URICollection (i.e. chained).

at(i)

Returns a clone of the URIjs instance at index i.

Parameters
  • i: Number

    The index of the instance to clone.

Returns
  • URI

    A clone of the corresponding URIjs instance from the collection.

clone()

Returns a new URICollection instance containing clones of the same URIs.

Parameters
  • None.
Returns
  • URICollection

    A new URICollection.

contains(val)

Predicate function indicating whether the collection contains the passed URI, given as a string, URIjs instance, or RegExp.

The comparison is against the array of #toString() results of the collection's elements.

If a URIjs instance is passed in as val, its own #toString() is used for comparison.

If a RegExp instance is passed in as val, the result is determined by calling its #.test() method with each collection element's #toString(), short-circuiting when any returns true.

Parameters
  • val: String, Object, RegExp

    The value to search for.

Returns
  • Boolean

    `true` if the collection contains the value; `false` otherwise.

countBy(countingMethod)

Accepts a property name or a function and uses it to create a plain Javascript object keyed by the outputs of the counting function, with values equal to the number of times a given output was returned.

Parameters
  • countingMethod: String, Function

    A string representing a method to invoke on each element, counting by the output, or a function which is passed each element in sequence and calculates the correct result to count by.

Returns
  • Object

    An object with keys determined by the counting criteria, with `Number` values corresponding to the number of times a given key was returned.

each(iterFn)

Accepts an iterator function and applies it to a cloned list of collection elements, returning a new URICollection instance with the results.

This is distinct from #map() because this implementation of #each() always returns a new URICollection, even if iterFn doesn't have any return value. #map() only returns a new URICollection if it results in an array of URIjs instances.

No changes are made to the original collection elements.

Parameters
  • iterFn: Function

    The iterator function which will be called with each collection element in sequence.

Returns
  • URICollection

    A URICollection instance created by applying `iterFn` to each of a cloned array of internal `URIjs` instances.

every(truthTest)

Returns true if every element passes a truth test passed in as a callback.

Uses Underscore's _.every, but guarantees immutability.

Parameters
  • truthTest: Function

    A function which is handed collection elements in sequence and should return true or false depending on the element's attributes.

Returns
  • Boolean

    True if every element passes the test; false if any element fails.

filter(predicateFn)

Just like the Underscore _.filter method, but returns a new URICollection with cloned URI instances.

Accepts a predicate function predicateFn and applies it to each element in sequence, returning a collection containing all elements that cause the predicate to return true.

Parameters
  • predicateFn: Function

    A function which will be handed each collection element and should return `true` for all elements to be included in the new collection.

Returns
  • URICollection

    A new URICollection instance containing the matching elements, regardless of the number of matches.

find(testFn)

Accepts a truth-testing function and returns the first element to cause a true return value. Proxies to Underscore's _.find, but returns a cloned URIjs instance to ensure immutability of collection elements.

Parameters
  • testFn: Function

    The truth-testing function which will iterate over collection elements, short-circuiting at the first to return true.

Returns
  • URI, null

    If a result is found, a clone of the matching `URIjs` instance. Otherwise, returns `null`.

findWhere(queryObject)

Accepts a query object and returns the first collection element with properties matching the key-value pairs in that object. Like #where(), but only returns the first result and does not wrap it in a collection - it returns a URIjs instance.

Parameters
  • queryObject: Object

    A plain Javascript object containing properties to match, e.g. `{domain: 'google.com'}`

Returns
  • URI

    A clone of the first `URIjs` element in the collection matching the query.

first(n)

Returns the first n list elements.

Parameters
  • n: Number

    The number of elements to return, defaulting to 1.

Returns
  • URICollection, URI

    If `n > 1`, returns a `URICollection`. If `n===1` or `typeof n === 'null'`, returns a cloned URIjs instance.

groupBy(groupingMethod, asStrings)

Accepts a property name or a function and uses it to create a plain Javascript object with the collection's elements keyed by the results of the grouping function or property.

Parameters
  • groupingMethod: String, Function

    A string representing a method to invoke on each element, grouping by the output, or a function which is passed each element in sequence and calculates the correct group key.

  • asStrings: Boolean

    (Optional) If `true` is passed, the value for each group key on the returned object will be an array of URI strings instead of URICollections.

Returns
  • Object

    An object with keys determined by the grouping criteria, with URICollections as the values. If `asStrings=true`, the values are converted to arrays of URI strings.

initial(n)

Returns a new URICollection with copies of the elements from from index 0 up to index #length() - n.

Parameters
  • n: Number

    The number of ending elements to omit, defaulting to 1.

Returns
  • URICollection

    A URICollection instance, even if only one element is returned. This is in line with Underscore's behavior.

invoke(methodName, args)

Accepts a method name and optional additional arguments to pass to that method, and invokes the given method on each URIjs instance in the collection.

Because URIjs methods without argument return attribute information - e.g. uri.domain() returns the string-format URI domain - a plain array of results is returned if no additional arguments are supplied.

Calling URIjs methods with arguments generally causes mutation and returns the object - e.g. uri.domain("justcats.net") returns the same object with its domain changed to justcats.net. For this reason, passing additional arguments causes the result to be wrapped in a new URICollection instance.

No changes are made to the original collection elements.

Parameters
  • methodName: String

    A string representing the method name to invoke on each collection element.

  • args: Array

    (Optional) Additional arguments to pass when invoking `methodName`.

Returns
  • Array, URICollection

    If no additional arguments are passed, a plain array of results. Otherwise, a new URICollection wrapping clones of the URIjs elements from the original collection.

last(n)

Returns the last n list elements.

Parameters
  • n: Number

    The number of elements to return, defaulting to 1.

Returns
  • URICollection, URI

    If `n > 1`, returns a `URICollection`. If `n===1` or `typeof n === 'null'`, returns a cloned URIjs instance.

map(mapFn)

Accepts an iterator function and applies it to a cloned list of collection elements. Depending on the iterator's output, this method will either return a new URICollection instance or a plain array.

If the mapFn returns an array of URIjs instances, they will be wrapped in a new URICollection and returned. If it returns another type of result, the plain array will be returned to the client.

A given map operation's outputs are assumed to be homogeneous with respect to type, so the type of the internal array resulting from proxying to _.map(list, mapFn) is determined by testing only the first element.

No changes are made to the original collection elements.

Parameters
  • mapFn: Function

    The iterator function which will be called with each collection element in sequence.

Returns
  • Array, URICollection

    Either a new `URICollection` (if an array of `URIjs` instances are returned) or a plain array of results (e.g. if strings of element hostnames are returned.)

pluck(propertyName)

Accepts a property name and returns a plain array with the value of that property for each element in the collection.
Similar to Underscore's own _.pluck but internally uses _.resolve in a map operation, because URIjs properties are all returned by methods.

Parameters
  • propertyName: String

    A string representing the property to pluck from each collection element.

Returns
  • Array

    A plain array of results.

push(linkList)

Alias for #add()

Parameters
  • linkList: String, Object, Array

    A single link, multiple links, or an array of links. A given link can be an existing URIjs instance or a string, which will be converted into a URIjs instance.

Returns
  • URICollection

    A reference to the same URICollection (i.e. chained).

reduce(iterator, initial)

Typical reduce operation, creating a single result by iterating over a list of elements from left to right.

Proxies to Underscore's _.reduce with an added safety for immutability.

Parameters
  • iterator: Function

    A function which takes `(soFar, elem)` as its arguments, where `soFar` is the value of the reduce operation up to that point and `elem` is the current collection element. It receives collection elements in left to right order.

  • initial: Any

    Initial value to pass as `current` to the first cycle of the iterator function.

Returns
  • Any

    Any value returned by the reduce operation. There is no automatic wrapping in new URICollection instances.

reduceRight(iterator, initial)

Typical reduce operation but reversed, creating a single result by iterating over a list of elements from right to left.

Proxies to Underscore's _.reduce with an added safety for immutability.

Parameters
  • iterator: Function

    A function which takes `(soFar, elem)` as its arguments, where `soFar` is the value of the reduce operation up to that point and `elem` is the current collection element. It receives collection elements in right to left order.

  • initial: Any

    Initial value to pass as `current` to the first cycle of the iterator function.

Returns
  • Any

    Any value returned by the reduce operation. There is no automatic wrapping in new URICollection instances.

reject(predicateFn)

Just like the Underscore _.reject method (the inverse of _.filter), but returns a new URICollection with cloned URI instances.

Accepts a predicate function predicateFn and applies it to each element in sequence, returning a collection containing all elements that cause the predicate to return false.

Parameters
  • predicateFn: Function

    A function which will be handed each collection element and should return `false` for all elements to be included in the new collection.

Returns
  • URICollection

    A new URICollection instance containing the matching elements, regardless of the number of matches.

rest(n)

Returns a new URICollection with copies of the elements from index n to the end.

Parameters
  • n: Number

    The index to start copying from, defaulting to 1.

Returns
  • URICollection

    A URICollection instance, even if only one element is returned. This is in line with Underscore's behavior.

sample(n)

Returns n randomly selected elements from the collection. If n==1 or typeof n === "null", returns a URIjs instance. If n > 1, returns a new URICollection instance.

Parameters
  • n: Number

    The number of elements to return in the sample, defaulting to 1.

Returns
  • URICollection, URI

    A URICollection instance if `n > 1`, or a `URIjs` instance otherwise.

shuffle()

Just like the Underscore _.shuffle method, but returns a new URICollection with cloned URI instances.

The new collection has its elements randomly shuffled.

Parameters
  • None.
Returns
  • URICollection

    A new URICollection instance containing clones of the shuffled elements.

size()

Returns the number of URI elements in the collection.

Parameters
  • None.
Returns
  • Number

    The size of the collection.

some(truthTest)

Returns true if any element passes a truth test passed in as a callback.

Uses Underscore's _.some, but guarantees immutability.

Parameters
  • truthTest: Function

    A function which is handed collection elements in sequence and should return true or false depending on the element's attributes.

Returns
  • Boolean

    True if any element passes the test; false if none of them do.

sortBy(sortingMethod)

Accepts a method name or a function and uses it to create a new URICollection with elements ordered by the function output or method invocation output for each object.

Parameters
  • sortingMethod: String, Function

    A string representing a method to invoke on each element, returning a value used to sort by, or a function which is passed each element in sequence and calculates the correct value to sort by.

Returns
  • URICollection

    A new collection with elements sorted according to the returned sorting criteria.

stringAt(i)

Returns the #toString() result of the URIjs instance at index i.

Parameters
  • i: Number

    The index of the instance to call `#toString()` on.

Returns
  • String

    The string returned from calling `#toString()` on the instance.

toArray()

Returns a plain array of cloned URIjs instances matching those in the collection.

Parameters
  • None.
Returns
  • Array

    An array of URIjs instances.

toString()

Returns an array of URI strings created by calling #toString() on each URIjs instance in the collection.

Parameters
  • None.
Returns
  • Array

    An array of URI strings.

where(queryObject)

Accepts a query object and returns all collection elements with properties matching the key-value pairs in that object.

Parameters
  • queryObject: Object

    A plain Javascript object containing properties to match, e.g. `{domain: 'google.com'}`

Returns
  • URICollection

    A new URICollection instance.
